## Why This Project

LangChain v1.0 introduced significant changes compared to earlier versions.
Instead of relying on theory alone, this repository documents my learning by
building and testing each concept hands-on using real code.

This helps solidify:
- How LangChain components interact
- How agents and tools are structured
- How prompts and models can be controlled dynamically
